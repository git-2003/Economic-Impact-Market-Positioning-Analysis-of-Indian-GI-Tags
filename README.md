# Economic-Impact-Market-Positioning-Analysis-of-Indian-GI-Tags
This project presents an interactive dashboard built using Microsoft Power BI to analyze the economic impact, market positioning, and distribution of Indian Geographical Indication (GI) Tags.
The dashboard provides a clean and interactive view of how GI tags are distributed across states, categories, market types, and economic impact levels. The objective is to transform raw GI data into actionable visual insights through effective dashboard design and data modeling.
